PASOS RESTANTES — Después de activar Authentication
====================================================

PASO 1 · Crear Firestore (base de datos)
-----------------------------------------
1. Abre: https://console.firebase.google.com/project/bank-minibanco/firestore
2. Clic en "Crear base de datos" / "Create database"
3. Modo: "Iniciar en modo de prueba" (30 días, ideal para desarrollo)
4. Región: southamerica-east1 (São Paulo) o la más cercana
5. Clic en "Crear" / "Enable"

PASO 2 · Pegar reglas de seguridad
-----------------------------------
1. En Firestore → pestaña "Reglas" / "Rules"
2. Borra todo el contenido actual
3. Copia y pega el archivo: firebase/firestore.rules
4. Clic en "Publicar" / "Publish"

PASO 3 · Probar la app
-----------------------
1. En terminal: npm run dev
2. Abre http://localhost:5173 (o el puerto que indique Vite)
3. Regístrate con:
   - Usuario 1: usuario1@test.com / Test1234
   - Usuario 2: usuario2@test.com / Test1234
4. Verifica en Firestore → Datos:
   - Colección "users" con saldo 100000
5. Con Usuario 1, transfiere $10000 a usuario2@test.com
6. Verifica:
   - Saldo de Usuario 1 baja a $90000
   - Saldo de Usuario 2 sube a $110000
   - Colección "movimientos" con el registro

PASO 4 · Si sale error de permisos
-----------------------------------
- Revisa que las reglas estén publicadas (Paso 2)
- Cierra sesión y vuelve a entrar
- Recarga la página (F5)

PASO 5 · Siguiente: subir a GitHub
-----------------------------------
- El archivo .env NO se sube (está en .gitignore)
- Solo se sube .env.example vacío
